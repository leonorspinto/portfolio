Migdalo APP

an application design

This is a very important project in my student journey. It was the last project I developed in highschool, it being the main one of those 3 years. We were told to do whatever we wanted for this project so I decided to design an application prototype for Migdalo.

Migdalo was a family business that cultivated and sold almonds. This way, I was able to explore an area that I had interest in and, at the same time, make it personal.

For this project, I used Adobe XD to design the layout of the APP and some functionalities - we didn't have a lot of time to develop this project, so I wasn't able to fully develop the app.

However, I was able to develop the full prototype of the app.

In this app, although in Portuguese, you are able to buy the products from the company, create an account and chat with other users and buyers about experiences and reviews, check out delicious recipes that include almonds, know more about the company and the almond itself, read some news about the topic and a lot more interesting things.
